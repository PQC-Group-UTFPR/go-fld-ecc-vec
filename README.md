# go-fld-ecc-vec
Go bindings for https://github.com/armfazh/fld-ecc-vec (private temporarily)

## Getting Started

First, clone the `go-fld-ecc-vec` repository:

```bash
$ git clone git@github.com:PQC-Group-UTFPR/go-fld-ecc-vec.git
$ cd go-fld-ecc-vec
```

Build the external library:

```bash
$ ./build.sh
```

### Examples

```bash
$ cd src/main
$ go build
$ ./main
```
